# Guion - Joel ( minutos)
## Utilidades + BD tiempo real + Agrupación de interfaces + README

Yo fui responsable de los módulos de utilidades, la base de datos en tiempo real, la agrupación de interfaces y la documentación del proyecto.

El generador de IDs asegura que nunca se repitan. La utilidad de tiempo calcula duraciones, formatea tiempos y aplica la política de redondeo hacia arriba usando math.ceil().

También implementé el guardado inmediato en JSON y la detección automática de cambio de día, creando archivos separados para cada fecha.

Finalmente, centralicé la exportación de las ventanas y documenté el proyecto en el README con instrucciones de instalación y uso.
