# --- DEPENDENCIAS DE PYTHON ---
# Este proyecto usa exclusivamente la biblioteca estándar de Python 3.
# NO requiere instalación de paquetes externos vía pip.
#
# Versión mínima requerida: Python 3.8 o superior
#
# Para verificar tu versión de Python:
#   python --version

# --- DEPENDENCIAS DEL SISTEMA ---
#
# Windows:
#   - Windows 10 o superior
#   - Cliente Telnet (opcional, para pruebas)
#   - Permisos de firewall para el puerto 5000
#
# Linux:
#   - NetworkManager (para crear hotspot)
#   - nmcli (comando incluido en NetworkManager)
#
# macOS:
#   - No requiere software adicional


# ============================================================
# GUÍA RÁPIDA DE EJECUCIÓN
# ============================================================

# --- PASO 1: INICIAR EL SERVIDOR ---
# En la laptop que actuará como servidor, abre una terminal y ejecuta:
#
#   python src/server_chat.py --port 5000
#
# El servidor escuchará en 0.0.0.0:5000 y mostrará su IP local.
# Anota esa IP (ej. 192.168.1.65 o 192.168.137.1 si usas hotspot).
#
# Si el firewall de Windows pregunta, selecciona "Permitir acceso".


# --- PASO 2: CONECTAR EL CLIENTE ---
# En la OTRA laptop, abre una terminal y ejecuta:
#
#   python src/client_chat.py --host <IP_DEL_SERVIDOR> --port 5000
#
# Ejemplos reales:
#   python src/client_chat.py --host 192.168.1.65 --port 5000
#   python src/client_chat.py --host 192.168.137.1 --port 5000
#
# IMPORTANTE: El cliente NECESITA los parámetros --host y --port.
# Sin --host, intentará conectar a 127.0.0.1 (localhost) y fallará
# si el servidor está en otra máquina.


# --- PASO 3: CHATEAR ---
# Una vez conectado, escribe mensajes y presiona Enter.
# Para salir del chat, escribe: salir


# --- PASO 4: TRANSFERENCIA DE ARCHIVOS (MEJORA OPCIONAL) ---
# Limitación: Solo archivos menores a 10 MB.
#
# Desde el cliente, usa el comando /enviar seguido de la ruta del archivo:
#
#   /enviar C:\Users\TuNombre\Desktop\documento.pdf
#
# También puedes ARRASTRAR el archivo directamente a la terminal CMD
# y la ruta se pegará automáticamente:
#
#   /enviar "C:\Users\TuNombre\Desktop\arrastrado_aqui.pdf"
#
# El servidor:
#   1. Recibirá el archivo
#   2. Lo guardará en results/received/
#   3. Calculará el hash SHA-256
#   4. Verificará la integridad
#   5. Responderá OK_VERIFIED o ERR_HASH

# ============================================================
# SOLUCIÓN DE PROBLEMAS COMUNES
# ============================================================

# PROBLEMA: ConnectionRefusedError / Conexión rechazada
# CAUSA: El servidor no está ejecutándose o el firewall bloquea el puerto.
# SOLUCIÓN:
#   1. Verifica que el servidor esté corriendo en la otra laptop.
#   2. Asegúrate de que el firewall permita el puerto 5000.
#   3. Prueba con hotspot en lugar de misma red Wi-Fi.

# PROBLEMA: Connection timed out / Tiempo de conexión agotado
# CAUSA: La IP del servidor es incorrecta o está en otra red.
# SOLUCIÓN:
#   1. Ejecuta ipconfig en la laptop servidora y verifica su IP.
#   2. Asegúrate de que ambas laptops estén en la misma red.

# PROBLEMA: FileNotFoundError al usar /enviar
# CAUSA: La ruta del archivo tiene espacios o caracteres especiales.
# SOLUCIÓN:
#   Encierra la ruta entre comillas:
#     /enviar "C:\Users\Mi Nombre\archivo.pdf"
#   O arrastra el archivo directamente a la terminal CMD.

# PROBLEMA: Archivo mayor a 10 MB no se transfiere
# CAUSA: Limitación del protocolo implementado.
# SOLUCIÓN:
#   Comprime el archivo o divídelo en partes menores a 10 MB.
